## 3. 修正固定對話框遮住可點擊畫面

- 移除 `StoryPuzzleGamePage` 原本置中的 925×440 固定 console。
- 點擊物件才顯示半透明浮動卡片。
- 卡片會依點擊物件位置出現在畫面另一側，避免蓋住正在調查的物件。
- 點擊卡片以外的場景，自動關閉。
- 按 Esc，自動關閉。
- 點擊另一個互動物件時，舊卡片先關閉，再顯示新內容。
- 答案錯誤、答案正確、提示、離開確認都使用相同浮動卡片。
